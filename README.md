# Bolt Wiki Repository

This is the repo for the wiki hosted at [https://wiki.vtbolt.com/](https://wiki.vtbolt.com/).

## Editing

Do not make changes to this html file directly. The website has built in editing and github support. If you want to add a page to the wiki, open the Administration tab in the sidebar and select `Editing` for a guide. There is also now a video guide on the `Editing` page.
